# Rachai

[cite_start]O **RachAl** é uma plataforma desenvolvida para facilitar a organização financeira entre amigos e colegas[cite: 8]. [cite_start]O foco é registrar gastos compartilhados, dividir valores e utilizar algoritmos para simplificar a quitação de dívidas[cite: 7, 9].

## 🚀 Funcionalidades Planejadas
* [cite_start]**Gestão de Perfil:** Criação e edição de contas de usuário[cite: 10, 11, 12].
* [cite_start]**Gestão de Grupos:** Criação de grupos e consulta de dívidas[cite: 13, 14, 16].
* [cite_start]**Registro de Despesas:** Divisão igual ou personalizada[cite: 18].
* [cite_start]**Inteligência:** Algoritmo de compensação automática e sugestão de eventos[cite: 19, 20].

## 🛠️ Tecnologias
* [cite_start]**Front-end:** React (Vite), HTML5, CSS3, JavaScript[cite: 33, 34, 35, 37].
* [cite_start]**Back-end:** Java com Spring Boot[cite: 38, 39, 40].
